# Avok Frontend

Frontend scaffold for the Avok escrow platform, built with Next.js App Router, Tailwind CSS, React Query, and Zustand.

## Expected API

Set the backend base URL with:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000/api/v1
```

## Planned flows

- Buyer dashboard
- Seller dashboard
- Admin dispute review dashboard
- Login and registration
- Payment initiation view
- Notification center

## Install and run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
